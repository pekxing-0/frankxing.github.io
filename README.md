# frankxing.github.io

Personal homepage of Frank Xing (邢志明).

Built with plain HTML and CSS — minimal, fast, no frameworks.

## Sections

- **Bio** — About Frank Xing
- **Career** — Professional timeline
- **Writing** — Articles from 老邢笔记 WeChat account
- **Contact** — LinkedIn, WeChat, THANK, Podcast

## Tech

- Pure HTML + CSS
- Inter font from Google Fonts
- No JavaScript frameworks
- Zero dependencies

## Deployment

Push to `main` branch of `pekxing-0/frankxing.github.io`. GitHub Pages will serve from `https://frankxing.github.io/`.
